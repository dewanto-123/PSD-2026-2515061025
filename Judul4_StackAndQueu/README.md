# Program Sistem Antrian Rumah Sakit Yang Menggunakan Queue Linked List

# Deskripsi Singkat
Program ini dibuat untuk mensimulasikan antrian pasien di rumah sakit dengan memanfaatkan struktur data Queue berbasis Linked List. Dimana program ini berfungsi mengelola urutan kedatangan pasien yang akan dilayani sesuai dengan aturan FIFO (First In First Out), yaitu pasien yang tiba terlebih dahulu mendapatkan pelayanan lebih awal. Pemilihan struktur data Queue dinilai tepat, karena mampu merepresentasikan antrian nyata seperti di loket rumah sakit, kasir, atau layanan publik lainnya. Sementara itu, implementasi Linked List memberikan keunggulan berupa kemampuan antrian untuk membesar atau mengecil secara dinamis tanpa adanya batasan kapasitas maksimum, berbeda dengan array yang bisa dibilang bersifat statis.

Program dimulai dengan pendefinisian kelas Node yang berfungsi untuk merepresentasikan setiap pasien dalam antrian. Di dalam konstruktor __init__ yang menerima parameter nama, data nama pasien disimpan ke dalam atribut self.nama, sedangkan atribut self.next diinisialisasi dengan None sebagai penunjuk ke node berikutnya pada linked list. Selanjutnya, didefinisikan kelas QueueRumahSakit sebagai struktur utama antrian. Pada konstruktor kelas ini, dua variabel self.front dan self.rear dibuat dengan nilai awal None yang masing‑masing menunjuk ke bagian depan dan belakang antrian. Kelas ini memiliki metode is_empty() yang mengembalikan True jika front bernilai None, menandakan antrian kosong, dan False jika sebaliknya.

Metode ambil_antrian(nama) digunakan untuk menambahkan pasien baru ke dalam antrian. Pertama, metode ini membuat node baru dari kelas Node dengan nama pasien. Kemudian dilakukan pengecekan apakah antrian kosong menggunakan is_empty(). Jika kosong, maka front dan rear sama‑sama menunjuk ke node baru tersebut. Jika tidak kosong, node baru disambungkan di belakang antrian dengan cara self.rear.next = new_node, lalu rear dipindahkan ke node baru. Setelah itu, program menampilkan pesan bahwa pasien berhasil masuk antrian rumah sakit.

Metode layani_pasien() berfungsi untuk menghapus dan melayani pasien yang berada di paling depan antrian. Pertama, metode memeriksa apakah antrian kosong; jika kosong, akan ditampilkan pesan “Tidak ada pasien dalam antrian” dan fungsi berhenti dengan return. Jika tidak kosong, nama pasien dari front disimpan ke variabel pasien kemudian ditampilkan pesan bahwa pasien tersebut sedang dilayani. Selanjutnya, pointer front dipindahkan ke node berikutnya (self.front = self.front.next), sehingga pasien paling depan terhapus. Apabila setelah penghapusan front bernilai None (artinya antrian menjadi kosong), maka rear juga diatur menjadi None untuk menjaga konsistensi.

Metode lihat_antrian() digunakan untuk menampilkan seluruh pasien yang sedang mengantri. Metode ini terlebih dahulu memeriksa apakah antrian kosong; jika kosong, muncul pesan “Antrian kosong”. Jika tidak kosong, dicetak judul “Daftar antrian pasien:”. Sebuah variabel current diisi dengan front untuk memulai penelusuran dari depan, dan variabel nomor diberi nilai 1. Selama current tidak bernilai None, program mencetak nomor antrian dan nama pasien, lalu current digeser ke node berikutnya dan nomor bertambah satu. Proses ini berulang hingga semua pasien dalam antrian ditampilkan.

Fungsi main() merupakan program utama yang pertama kali membuat objek rs dari kelas QueueRumahSakit. Di dalam perulangan while True, program menampilkan menu utama yang terdiri dari empat pilihan: (1) Ambil Antrian, (2) Layani Pasien, (3) Lihat Antrian, dan (4) Keluar Program. Pengguna diminta memasukkan pilihan melalui input(). Jika pilihan adalah “1”, program meminta nama pasien lalu memanggil metode ambil_antrian(nama). Jika pilihan “2”, dipanggil layani_pasien(). Jika pilihan “3”, dipanggil lihat_antrian(). Jika pilihan “4”, program mencetak pesan “Program selesai. Terima kasih!” lalu keluar dari perulangan dengan break. Apabila input tidak sesuai dengan pilihan 1–4, program mencetak “Pilihan tidak valid”. Terakhir, di luar fungsi main(), terdapat blok if __name__ == "__main__": yang memastikan fungsi main() hanya dijalankan ketika file dieksekusi secara langsung, bukan ketika diimpor sebagai modul.

Output program Sistem Antrian Rumah Sakit dimulai ketika pengguna memilih menu 1 untuk mengambil antrian, lalu memasukkan nama Dewa sehingga program menampilkan pesan "Dewa berhasil masuk antrian rumah sakit". Kemudian pengguna memilih menu 1 lagi dan memasukkan nama Arkan, yang langsung ditambahkan ke belakang antrian dengan pesan konfirmasi serupa. Setelah itu, pilihan menu 3 digunakan untuk melihat daftar antrian, dan program menampilkan dua pasien berurutan yaitu 1. Dewa dan 2. Arkan. Selanjutnya pengguna memilih menu 2 untuk melayani pasien; karena Dewa berada di posisi depan, program menampilkan "Pasien Dewa sedang dilayani" dan Dewa dihapus dari antrian. Ketika menu 3 dipilih kembali, antrian hanya menampilkan 1. Arkan. Pengguna kemudian mencoba memasukkan pilihan 7 yang tidak tersedia, dan program merespons dengan "Pilihan tidak valid". Terakhir, pilihan 4 dipilih untuk keluar, program menampilkan "Program selesai. Terima kasih!" dan kembali ke prompt terminal. Seluruh output ini membuktikan bahwa antrian bekerja sesuai prinsip FIFO, antrian dapat bertambah dan berkurang secara dinamis, serta program mampu menangani input yang salah dengan baik.
