# Program Sistem Antrian Rumah Sakit Yang Menggunakan Queue Linked List

# Deskripsi Singkat
Program ini dibuat untuk mensimulasikan antrian pasien di rumah sakit dengan memanfaatkan struktur data Queue berbasis Linked List. Dimana program ini berfungsi mengelola urutan kedatangan pasien yang akan dilayani sesuai dengan aturan FIFO (First In First Out), yaitu pasien yang tiba terlebih dahulu mendapatkan pelayanan lebih awal. Pemilihan struktur data Queue dinilai tepat, karena mampu merepresentasikan antrian nyata seperti di loket rumah sakit, kasir, atau layanan publik lainnya. Sementara itu, implementasi Linked List memberikan keunggulan berupa kemampuan antrian untuk membesar atau mengecil secara dinamis tanpa adanya batasan kapasitas maksimum, berbeda dengan array yang bisa dibilang bersifat statis.
