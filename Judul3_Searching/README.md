# Program Pencarian Data Mahasiswa Menggunakan Binary Search

# Deskripsi Singkat
Program ini adalah pencarian data mahasiswa menggunakan algoritma Binary Search. Data disimpan sebagai daftar tupel (nama, NIM) yang sudah terurut berdasarkan nama. Pengguna memasukkan nama yang dicari, lalu program akan membagi daftar menjadi dua bagian secara berulang hingga nama ditemukan atau tidak ada. Jika ditemukan, program menampilkan nama beserta NIM-nya. Binary Search dipilih karena lebih cepat dibanding pencarian linear pada data terurut.

# Source Code

Program dimulai dengan mendefinisikan fungsi binary_search_mahasiswa(data, target) yang menerima daftar mahasiswa dan nama target, lalu menginisialisasi kiri = 0 sebagai batas kiri dan kanan = len(data) - 1 sebagai batas kanan, kemudian mengubah target menjadi huruf kecil dengan target.lower() agar pencarian tidak sensitif huruf besar kecil, lalu masuk ke perulangan while kiri <= kanan untuk memastikan masih ada ruang pencarian. Di dalam perulangan, indeks tengah dihitung dengan tengah = (kiri + kanan) // 2, kemudian nama mahasiswa di indeks tengah diambil dan diubah menjadi huruf kecil disimpan di nama_tengah, lalu dicetak nama yang sedang diperiksa. Jika nama_tengah sama dengan target, fungsi mengembalikan indeks tengah. Jika nama_tengah lebih kecil dari target, batas kiri digeser ke kiri = tengah + 1 untuk mencari di bagian kanan. Jika lebih besar, batas kanan digeser ke kanan = tengah - 1 untuk mencari di bagian kiri. Ketika perulangan selesai tanpa menemukan target, fungsi mengembalikan -1 sebagai tanda tidak ditemukan.

<img width="575" height="437" alt="Screenshot 2026-05-11 225801" src="https://github.com/user-attachments/assets/caf9a288-605e-4334-9f63-364ea540cdea" />
Fungsi main() dimulai dengan mendefinisikan daftar mahasiswa yang berisi tujuh tupel, masing-masing berisi nama dan NIM, misalnya ("Arkan", "231401001") sampai ("Gusrak", "231401007"), dimana daftar ini sudah terurut berdasarkan nama. Kemudian program mencetak header "Daftar Mahasiswa:" diikuti perulangan for nama, nim in mahasiswa: yang mencetak setiap nama dan NIM dengan format - Nama : NIM. Setelah itu, program meminta input dari pengguna dengan input("\nMaskukan nama mahasiswa yang ingin dicari: ") (ada kesalahan ketik "Maskukan" seharusnya "Masukkan") dan menyimpannya di variabel target. Selanjutnya, fungsi binary_search_mahasiswa(mahasiswa, target) dipanggil dan hasil indeksnya disimpan di variabel index. Jika index != -1 (artinya ditemukan), maka program mengambil tupel mahasiswa[index], memisahkan ke nama dan nim, lalu mencetak pesan {nama} ditemukan dengan NIM {nim}. Sebaliknya, jika index == -1, program mencetak "Nama tidak ditemukan".

Dua baris terakhir program, yaitu if __name__ == "__main__": diikuti main(), berfungsi untuk mengeksekusi fungsi main() hanya ketika file Python tersebut dijalankan langsung sebagai program utama, bukan ketika file diimpor sebagai modul ke file lain. Variabel __name__ akan bernilai "__main__" jika file dieksekusi langsung, sehingga kondisinya benar dan main() dipanggil. Ini adalah praktik umum dalam Python untuk memisahkan kode yang dapat dijalankan dari kode yang dapat digunakan ulang, sehingga memungkinkan fungsi-fungsi seperti binary_search_mahasiswa tetap bisa diimpor tanpa menjalankan antarmuka input pengguna saat diimpor.
